## Descrição

Este projeto foi desenvolvido como atividade prática da disciplina de Lógica de Programação.

O sistema foi construído utilizando:

* HTML5
* CSS3
* JavaScript

Cada questão foi organizada em uma página individual, permitindo a navegação através de uma página inicial (menu principal).

O projeto utiliza um tema escuro para melhorar a experiência visual e facilitar a leitura.

## Questões Desenvolvidas

### Questão 1

Leitura de três números inteiros e cálculo da média.

### Questão 2

Cálculo da área de uma parede e quantidade de tinta necessária para pintura.

### Questão 3

Cálculo do consumo de combustível e custo total de uma viagem.

### Questão 4

Cálculo da média de um aluno e verificação de aprovação ou reprovação.

### Questão 5

Identificação do tipo de triângulo:

* Equilátero
* Isósceles
* Escaleno

### Questão 6

Cálculo do Índice de Massa Corporal (IMC) e classificação da faixa de risco.

---

## Como Executar

1. Abra o arquivo `index.html` em um navegador.
2. Escolha uma das questões disponíveis no menu principal.
3. Clique em **Mostrar Resolução** para visualizar o resultado da questão.
